<h1> Overview </h1>
<p>This is a simple app that contain features:</p>
<p>1. Reward Points Redemption</p>
<p>2. Calculate and Deduct Reward Points</p>
<p>3. Earning/Saving Reward Points </p>

<h1> How to Run? </h1>
<p>1. Clone this repository.</p>
<p>2. Open the Solution.</p>
<p>3. Uncomment 1 of 3 features in Program.cs (Already divided by Region)</p>
<p>4. Press "ConsoleApp1" on the top side of Visual Studio</p>
